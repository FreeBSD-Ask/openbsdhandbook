# 数据库

数据库页面：

- [MariaDB](mariadb.md)
  在 OpenBSD 上安装并配置 MariaDB 关系型数据库服务器，作为 MySQL 的直接替代方案。
- [memcached](memcached.md)
  在 OpenBSD 上安装并配置 memcached，通过网络提供高性能的内存键值缓存。
- [PostgreSQL](postgresql.md)
  在 OpenBSD 上安装并配置 PostgreSQL，实现可靠的关系型数据管理，支持访问控制与 TLS。
- [Redis](redis.md)
  在 OpenBSD 上安装并配置 Redis，提供高性能键值存储，涵盖服务管理与访问安全。
