# 邮件

邮件服务器页面：

- [Dovecot](dovecot.md)
  在 OpenBSD 上的配置与使用。
- [Exim](exim.md)
  在 OpenBSD 上使用 Exim 作为灵活且完全可配置的邮件传输代理。涵盖安装、TLS 设置、本地与远程投递，以及常见配置示例。
- [Postfix](postfix.md)
  在 OpenBSD 上使用 Postfix 作为安全、灵活且广泛支持的邮件传输代理。涵盖安装、中继、TLS 与虚拟域支持。
- [Rspamd](rspamd.md)
  在 OpenBSD 上配置 Rspamd 进行垃圾邮件过滤与邮件分类，并提供与 smtpd 及其他 MTA 的集成示例。
- [smtpd](smtpd.md)
  在 OpenBSD 上使用 smtpd(8) 收发邮件，默认配置安全，支持中继、本地投递与 TLS。
