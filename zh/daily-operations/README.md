# 第二部分 日常操作

OpenBSD 手册第二部分介绍系统日常使用中的常见任务与工具，涵盖图形环境、打印、多媒体、兼容层以及其他常规活动。

本部分包括：

- [图形环境](graphical-environments.md)
  桌面环境与窗口管理器的配置与使用，包括显示管理器、字体和输入设备。
- [打印](printing.md)
  本地与网络打印机的设置、打印队列管理以及打印服务配置。
- [多媒体](multimedia.md)
  音频与视频配置、媒体播放以及硬件注意事项。
- [Linux 兼容性](linux-compatibility.md)
  使用 OpenBSD 的 Linux 模拟运行兼容的应用程序。
- [Windows 兼容性](windows-compatibility.md)
  通过网络协议和文件格式与 Windows 系统互操作。
- [游戏](games.md)
  OpenBSD 上游戏的可用情况、安装与配置。
