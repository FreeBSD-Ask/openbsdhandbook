# Part II. Daily Operations

Part II of the OpenBSD Handbook describes common tasks and tools for everyday use of the system. It covers graphical environments, printing, multimedia, compatibility layers, and other routine activities.

This section includes:

- [Graphical Environments](graphical-environments.md)
  Configuration and use of desktop environments and window managers, including display managers, fonts, and input devices.
- [Printing](printing.md)
  Setting up local and network printers, managing print queues, and configuring print services.
- [Multimedia](multimedia.md)
  Audio and video configuration, media playback, and hardware considerations.
- [Linux Compatibility](linux-compatibility.md)
  Using OpenBSD’s Linux emulation to run compatible applications.
- [Windows Compatibility](windows-compatibility.md)
  Interoperating with Windows systems through network protocols and file formats.
- [Games](games.md)
  Availability, installation, and configuration of games on OpenBSD.
