# How-to Guides

The how-to guides provide focused procedures for common OpenBSD tasks. Each guide favors direct commands, minimal assumptions, and configuration examples that can be adapted to local policy.

## Available Guides

- [Build a simple router and firewall](router-firewall.md)
  — configure routing, NAT, DHCP, and local DNS caching.
- [Install Z shell](zsh.md)
  — install and configure `zsh` for a user account.
- [Set up WordPress](wordpress.md)
  — deploy WordPress with httpd, PHP-FPM, and MariaDB.
