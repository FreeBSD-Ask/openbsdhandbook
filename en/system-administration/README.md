# Part III. System Administration

Part III of the OpenBSD Handbook covers administrative responsibilities involved in managing an OpenBSD system. It addresses system startup, localization, security practices, storage management, upgrades, and virtualization.

This section includes:

- [The OpenBSD Boot Process](boot-process.md)
  An overview of the system startup sequence, including the boot loader, kernel initialization, and multi-user mode.
- [Storage and File Systems](storage.md)
  Configuration and management of storage devices, disk partitioning, file systems, and related utilities.
- [Localization](localization.md)
  Configuring language, keyboard layouts, time zones, and locale-specific settings.
- [Security](security.md)
  System hardening, privilege management, authentication methods, and security auditing.
- [Updating and Upgrading](upgrade.md)
  Procedures for applying patches, performing minor updates, and upgrading to new releases.
- [Virtualization](virtualization.md)
  Running OpenBSD as a host or guest using supported virtualization technologies.
