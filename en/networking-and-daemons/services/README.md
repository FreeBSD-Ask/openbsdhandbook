# Services

The Services section groups daemon configuration and operational guidance into clear functional categories.

Categories:

- [Database](database-services/README.md)
  Database server deployment and administration.
- [Directory](directory-services/README.md)
  Directory and authentication infrastructure.
- [File](file-services/README.md)
  Network file services and sharing protocols.
- [File Synchronization](sync-services/README.md)
  Synchronization and mirroring utilities.
- [FTP Services](ftp-services/README.md)
  FTP daemons and related features for file transfer and provisioning.
- [Logging](logging-services/README.md)
  System logging facilities, collection, forwarding, and retention policies.
- [Mail](mail-services/README.md)
  Mail transfer, delivery, and filtering components.
- [Messaging](messaging-services/README.md)
  Message brokers and inter-service communication.
- [Monitoring](monitoring-services/README.md)
  Host and service monitoring, metrics collection, and alerting.
- [Name](name-services/README.md)
  Authoritative and recursive DNS services.
- [Networking Services](networking-services/README.md)
  Supporting daemons such as DHCP, router advertisements, and SLAAC.
- [Remote Access](remote-access-services/README.md)
  Secure remote administration and access.
- [Time](time-services/README.md)
  Network time synchronization and clock discipline.
- [Web](web-services/README.md)
  HTTP servers, TLS termination, and reverse proxying.
