# Part IV. Networking & Daemons

Part IV covers network configuration and the administration of daemons that provide core network services. It includes firewalls, serial communications, and service categories.

This section includes:

- [PF](pf/README.md)
  Overview of the OpenBSD packet filter and its role in traffic filtering, NAT, redirection, and logging. Detailed chapters are organized under the PF section itself.
- [Serial Communication](serial.md)
  Configuration and use of serial ports, terminals, and related utilities.
- [Services](services/README.md)
  Index of all service categories and their individual daemon guides.

Service categories:

- [Database](services/database-services/README.md)
  Database server configuration and administration.
- [Directory](services/directory-services/README.md)
  Directory and authentication services.
- [File](services/file-services/README.md)
  Network file services and sharing protocols.
- [File Synchronization](services/sync-services/README.md)
  Synchronization and mirroring utilities.
- [FTP Services](services/ftp-services/README.md)
  FTP daemons and related features for file transfer and provisioning.
- [Logging](services/logging-services/README.md)
  System logging with local facilities and remote forwarding.
- [Mail](services/mail-services/README.md)
  Mail transfer, delivery, and filtering.
- [Messaging](services/messaging-services/README.md)
  Message brokers and inter-service communication.
- [Monitoring](services/monitoring-services/README.md)
  Host and service monitoring and metrics.
- [Name](services/name-services/README.md)
  Authoritative and recursive DNS services.
- [Networking Services](services/networking-services/README.md)
  Supporting daemons for DHCP, router advertisements, and SLAAC.
- [Remote Access](services/remote-access-services/README.md)
  Secure remote administration.
- [Time](services/time-services/README.md)
  Network time synchronization and clock discipline.
- [Web](services/web-services/README.md)
  HTTP servers, TLS termination, and reverse proxying.
