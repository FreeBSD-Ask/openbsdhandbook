# Web 服务

Web 服务器页面：

- [Apache](apache.md)
  使用 www/apache 软件包在 OpenBSD 上安装并配置 Apache HTTP Server，支持 TLS 与虚拟主机。
- [httpd](httpd.md)
  使用 OpenBSD 原生的 `httpd(8)` 守护进程，以安全且简洁的配置通过 HTTP 与 HTTPS 提供静态与 FastCGI 内容。
- [nginx](nginx.md)
  在 OpenBSD 上安装并配置 nginx，用于静态内容、TLS、FastCGI 与反向代理。
- [relayd](relayd.md)
  在 OpenBSD 上使用 `relayd(8)` 进行 TLS 终止、反向代理与 HTTP 负载均衡，并集成监控与过滤。
