# 服务

服务章节将守护进程的配置与运维指南按功能分为若干清晰的类别。

类别：

- [数据库](database-services/README.md)
  数据库服务器部署与管理。
- [目录](directory-services/README.md)
  目录与认证基础设施。
- [文件](file-services/README.md)
  网络文件服务与共享协议。
- [文件同步](sync-services/README.md)
  同步与镜像工具。
- [FTP 服务](ftp-services/README.md)
  FTP 守护进程及相关文件传输与分发功能。
- [日志](logging-services/README.md)
  系统日志设施、收集、转发与保留策略。
- [邮件](mail-services/README.md)
  邮件传输、投递与过滤组件。
- [消息](messaging-services/README.md)
  消息代理与服务间通信。
- [监控](monitoring-services/README.md)
  主机与服务监控、指标收集与告警。
- [名称](name-services/README.md)
  权威 DNS 与递归 DNS 服务。
- [网络服务](networking-services/README.md)
  DHCP、路由器通告与 SLAAC 等支撑性守护进程。
- [远程访问](remote-access-services/README.md)
  安全的远程管理与访问。
- [时间](time-services/README.md)
  网络时间同步与时钟校准。
- [Web](web-services/README.md)
  HTTP 服务器、TLS 终止与反向代理。
