# 网络服务

网络服务页面：

- [OpenBGPD](openbgpd.md)
  在 OpenBSD 上使用 `bgpd` 设计与运行 BGP：外部对等、iBGP、过滤、团体属性与 RPKI。
- [DHCP](dhcp.md)
  配置 OpenBSD 的 DHCP 客户端与服务器，包括接口设置、租约管理与 IPv6 相关注意事项。
- [rad](rad.md)
  在 OpenBSD 上使用 `rad(8)` 为 SLAAC 提供 IPv6 路由通告，包括与 DHCP 及相关工具的对比。
- [slaacd](slaacd.md)
  在 OpenBSD 上使用 `slaacd(8)` 通过无状态地址自动配置（SLAAC）自动配置 IPv6 地址。
