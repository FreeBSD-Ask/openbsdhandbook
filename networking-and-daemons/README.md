# 第四部分 网络与守护进程

第四部分介绍网络配置，以及提供核心网络服务的守护进程的管理。内容涵盖防火墙、串口通信和服务分类。

本部分包含：

- [PF](pf/README.md)
  OpenBSD 包过滤器的概览，及其在流量过滤、NAT、重定向和日志记录方面的作用。详细章节组织在 PF 部分之内。
- [串口通信](serial.md)
  串口、终端及相关工具的配置与使用。
- [服务](services/README.md)
  所有服务类别及其各自守护进程指南的索引。

服务类别：

- [数据库](services/database-services/README.md)
  数据库服务器的配置与管理。
- [目录](services/directory-services/README.md)
  目录与认证服务。
- [文件](services/file-services/README.md)
  网络文件服务与共享协议。
- [文件同步](services/sync-services/README.md)
  同步与镜像工具。
- [FTP 服务](services/ftp-services/README.md)
  FTP 守护进程及用于文件传输与分发的相关功能。
- [日志](services/logging-services/README.md)
  使用本地设施与远程转发的系统日志。
- [邮件](services/mail-services/README.md)
  邮件传输、投递与过滤。
- [消息](services/messaging-services/README.md)
  消息代理与跨服务通信。
- [监控](services/monitoring-services/README.md)
  主机与服务监控及指标采集。
- [名称](services/name-services/README.md)
  权威与递归 DNS 服务。
- [网络服务](services/networking-services/README.md)
  DHCP、路由通告和 SLAAC 的辅助守护进程。
- [远程访问](services/remote-access-services/README.md)
  安全的远程管理。
- [时间](services/time-services/README.md)
  网络时间同步与时钟校准。
- [Web](services/web-services/README.md)
  HTTP 服务器、TLS 终止与反向代理。
